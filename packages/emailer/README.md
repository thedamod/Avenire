# @avenire/emailer

Email sending layer for Avenire.

## Responsibilities

- wraps provider logic (Resend)
- imports templates from `@avenire/emails`
- exposes app-facing send helpers

## Scripts

- `pnpm --filter @avenire/emailer check-types`
- `pnpm --filter @avenire/emailer lint`
- `pnpm --filter @avenire/emailer build`
