# @avenire/observability

Shared observability/logging utilities.

## Responsibilities

- centralize telemetry client setup
- expose helpers used by apps/services for structured logs and events

## Scripts

- `pnpm --filter @avenire/observability build`
- `pnpm --filter @avenire/observability check-types`
- `pnpm --filter @avenire/observability lint`
