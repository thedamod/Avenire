# @avenire/linting

Shared linting and formatting config package.

## Exports

- `@avenire/linting/biome` -> base Biome config JSON

## Scripts

- `pnpm --filter @avenire/linting lint`
- `pnpm --filter @avenire/linting format`
- `pnpm --filter @avenire/linting biome:check`
- `pnpm --filter @avenire/linting biome:format`
