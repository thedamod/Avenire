# @avenire/ui

Shared UI component and media package.

## Exports

- global styles (`./styles.css`)
- utility + hooks modules
- component entrypoints (`./components/*`)
- media components (`./media`)

## Scripts

- `pnpm --filter @avenire/ui build`
- `pnpm --filter @avenire/ui check-types`
- `pnpm --filter @avenire/ui lint`
